# DISCOGS DASHBOARD - BY BLAKE ELDRIDGE

Will update this later, but if you happen to find this github, then this is the start of a discogs dashboard using a golang backend combined with a react frontend using wails as the glue to pull them together.

idea is a convienient, open-source way to view your collection, add to it, add to a want list etc whilst being nice to the rate limits of discogs api and stuff.

so if you happen to want to help when the project is done, feel free.

### Feature aims

- grid + scroll view of whole collection
- view and edit individual collection items
- add custom ratings, price paid and information about each album (maybe ratings per song???)

- grid + scroll view for wantlist
- browsing section to add to the wantlist
- way to add to your collection to, from the wantlist or from the browse section

- dashboard displaying some usual stats
- estimated collection value
- total cost paid for collection
- number of albums, number of cds, vinyls etc

- cool cover art board of all your collection or something - seen that a bunch
